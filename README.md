# Responsive Portfolio Landing Page

A modern, fully responsive personal portfolio landing page. This project focuses on clean architecture, smooth user animations, and mobile-first responsiveness.

## 🛠️ Features & Technical Details
* **Clean Folder Architecture:** Organized structure with separated folders for `css`, `img`, and `js`.
* **Mobile Responsiveness:** Smooth layout adaptation across all devices using CSS Media Queries.
* **Interactive Elements:** JavaScript-powered components to enhance user experience and interaction.
* **Modern Design:** Polished UI components, optimized imagery, and typography.

## 💻 Tech Stack
* **HTML5:** Semantic document structure.
* **CSS3:** Flexible layouts and custom transitions.
* **JavaScript (JS):** Core logic for interactive components and menu behavior.
